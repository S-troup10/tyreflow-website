# TyreFlow Website

Premium bike tyres landing page with futuristic orange/black design.

## Deploy to GitHub Pages

1. Create GitHub repo: `tyreflow-website`
2. Push this code
3. Enable GitHub Pages in settings
4. Site live at: `https://yourusername.github.io/tyreflow-website`

## Local Development

Open `index.html` in browser or use Live Server.

## Shopify Integration

All "Shop Now" buttons link to: `https://tyreflow.myshopify.com`

## Colors

- Primary Orange: #FF6B35
- Orange Glow: #FF8C42
- Black: #0A0A0A
